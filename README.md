# 🎉 Discover Events - Plataforma de Eventos

Uma plataforma moderna e responsiva para descobrir e participar de eventos incríveis na sua região.

## 📋 Descrição

Discover Events é um site elegante e funcional que permite aos usuários explorar diversos tipos de eventos, desde conferências de tecnologia até festivais de música e exposições de arte. O site foi desenvolvido com HTML5 e CSS3, oferecendo uma experiência visual atraente e uma navegação intuitiva.

## 🎯 Características

### Interface Principal (`index.html`)
- **Header Atraente**: Com branding visual e navegação clara
- **Grid de Eventos**: Exibição responsiva de eventos em cards
- **Informações Detalhadas**: Data, hora, local e descrição de cada evento
- **Botões de Ação**: "Saiba Mais" e "Inscrever-se" para cada evento
- **Responsividade**: Design que se adapta perfeitamente a todos os tamanhos de tela

### Páginas de Eventos
Cada evento possui uma página dedicada com:
- **Informações Completas**: Data, hora, local e preço de ingresso
- **Descrição Detalhada**: Sobre o evento e sua importância
- **Destaques**: Principais atrações e diferenciais
- **Informações Práticas**: Como chegar, acessibilidade, políticas
- **Call-to-Action**: Botões para compra de ingressos e compartilhamento

## 🎨 Design e Estilo

### Palheta de Cores
- **Gradiente Principal**: #667eea para #764ba2 (roxo/azul)
- **Fundo**: Branco para cards
- **Texto**: Cinzento escuro (#333)
- **Acentos**: Roxo (#667eea)

### Tipografia
- **Font**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- Responsiva em todos os tamanhos

## 📱 Responsividade

- **Desktop**: Layout full com grid de 3 colunas
- **Tablet**: Grid adaptativo de 2 colunas
- **Mobile**: Layout em coluna única com botões empilhados

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Navegue entre os eventos usando os links do menu
3. Clique em "Saiba Mais" para ver detalhes completos de cada evento
4. Use o botão "Voltar" para retornar à página inicial
